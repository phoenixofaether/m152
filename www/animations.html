CSS Animations

Keyframes

CSS allows animation of HTML elements without using JavaScript or Flash!
When you specify CSS styles inside the @keyframes rule, the animation will gradually change from the current style to the new style at certain times.
To get an animation to work, you must bind the animation to an element. The following example binds the "example" animation to the

element. The animation will last for 4 seconds, and it will gradually change the background-color of the
element from "red" to "yellow": 

          
            /* The animation code */
            @keyframes example {
              from {background-color: red;}
              to {background-color: yellow;}
            }

            /* The element to apply the animation to */
            div {
              width: 100px;
              height: 100px;
              background-color: red;
              animation-name: example;
              animation-duration: 4s;
              animation-iteration-count: 5;
            }

It is also possible to define what should change at which percantage of completed animation.
For example you can say, that after half a rotation the color of the element changes to red.


          
            /* The animation code */
            @keyframes animationwidthExample {
              10% { background-color: orange; }
              50% { background-color: green; transform: rotate(180deg); }
              75% { background-color: black; transform: rotate(90deg); }
              100% { transform: rotate(180deg); }
            }

Transforms

The transform CSS property lets you rotate, scale, skew, or translate an element. It modifies the coordinate space of the CSS. An example for that can be seen in the code above.

Modifying Svg-Images with keyframes

As a svg image isn't saved in pixels, but in vectors, it allows us to modify these vectors (here paths) with CSS or JavaScript.
First you need to save your SVG-Image as SVG code and copy it into your website. This allows you to access the seperate vectors/parts of the SVG-Image.
Next you need to add classes to the parts of the SVG-Image that you want to modify/access.
Finally you can add CSS animations to those classes. 

          
            <svg shape-rendering="geometricPrecision" text-rendering="geometricPrecision" width="400" height="100">
              <!--Black background-->
              <rect class="svg-background" width="400" height="100"/>
              <!--"Example" Text-->
              <text class="svg-text svg-text1" font-family=""Roboto"" font-size="28" font-weight="400" transform="translate(19.888395 59.863499)" fill="#fff">
                <tspan y="0" font-weight="400"><![CDATA[Example ]]></tspan>
              </text>
              <!--"for" Text-->
              <text class="svg-text svg-text2" font-family=""Roboto"" font-size="28" font-weight="400" transform="translate(138.825946 59.795475)" fill="#fff">
                <tspan y="0" font-weight="400"><![CDATA[for]]></tspan>
              </text>
              <!--"SVG-Animation" Text-->
              <text class="svg-text svg-text3" font-family=""Roboto"" font-size="28" font-weight="400" transform="translate(186.952897 59.795475)" fill="#fff">
                <tspan y="0" font-weight="400"><![CDATA[SVG-Animation]]></tspan>
              </text>
            </svg>

Html Canvas

Html Canvas are used to draw graphics with JavaScript. With the element <canvas> you can create a empty container that can be "filled" with JavaScript.

            
              <canvas id="myCanvasExample" width="200" height="100" style="border:1px solid #000000;">
              </canvas>
              <script>
                var c = document.getElementById("myCanvasExample");
                var ctx = c.getContext("2d");
                ctx.beginPath();
                ctx.arc(95, 50, 40, 0, 2 * Math.PI);
                ctx.stroke();
                </script>

Scroll-Animations

Scroll Animations are a kind of animation that takes place, while the visiter scrolls up and down. Those animations require a combination between html css and javaScript, but the main thing that triggers the animation is the javaScript part.

              
                function reveal() {
                  var reveals = document.querySelectorAll(".reveal"); // gets class
                  for (var i = 0; i < reveals.length; i++) {
                      var windowHeight = window.innerHeight; // current hight
                      var elementTop = reveals[i].getBoundingClientRect().top; // top of .reveal element
                      var elementVisible = 20; // how much of the element needs to be visible (in px)
                      if (elementTop < windowHeight - elementVisible) {
                          reveals[i].classList.add("active"); // adds active to classlist
                      } else {
                          reveals[i].classList.remove("active"); // removes active to classlist
                      }
                  }
                }
              
                window.addEventListener("scroll", reveal); // launch the reveal function on every scroll
              
                reveal(); // check current position
